# Medical Triage RL Triples

Dataset of 63 `(observation, action, reward)` triples from the **Medical Triage OpenEnv** environment — a clinical AI benchmark for the Meta × Scaler PyTorch Hackathon 2026.

Each record represents one episode: an AI agent reads a patient case (observation), produces a clinical decision (action), and receives a graded reward based on clinical correctness.

## Environment

- **Space:** [`kunalkachru23/medical-triage-env`](https://huggingface.co/spaces/kunalkachru23/medical-triage-env)
- **Grading:** Deterministic, using NHS NEWS2 and PEWS protocols
- **Reward range:** `(0.0001, 0.9999)` — strict open interval (OpenEnv spec)

## Fields

| Field | Type | Description |
|-------|------|-------------|
| `task_id` | string | One of 8 clinical tasks |
| `case_id` | string | Unique patient case identifier (e.g. `ST001`, `PD003`) |
| `patient_history` | string | Full clinical presentation text shown to the agent |
| `task_description` | string | Agent instruction prompt |
| `action` | object | Agent response JSON |
| `reward` | float | Scalar reward in (0.0001, 0.9999) |
| `score_breakdown` | object | Per-dimension score breakdown |
| `feedback` | string | Clinical rationale from the grader |
| `model` | string | Agent type used (e.g. `mock_rule_based`) |

## Tasks (8 total, 63 cases)

| Task | Cases | Difficulty | Clinical Focus |
|------|-------|------------|----------------|
| `simple_triage` | 10 | Easy | NHS NEWS2 scoring, priority classification |
| `conflicting_vitals` | 8 | Medium | Misleading vital signs, correct critical sign identification |
| `masked_deterioration` | 10 | Hard | Drug-masked deterioration (beta-blockers, steroids, NSAIDs) |
| `demographic_fairness` | 12 | Medium | Identical vitals, different demographics — bias detection |
| `deteriorating_patient` | 7 | Hard | Multi-turn escalation timing |
| `sepsis_bundle` | 4 | Hard | Surviving Sepsis Campaign Hour-1 bundle compliance |
| `paediatric_triage` | 6 | Hard | PEWS age-specific thresholds (infant → adolescent) |
| `medication_reconciliation` | 6 | Hard | Drug interactions, dosing errors, allergy conflicts |
