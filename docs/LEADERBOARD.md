## Key Observations

### Format compliance is a hard prerequisite
DeepSeek-R1-Distill-Qwen-32B scores **0.000** on 6 of 11 tasks — not because it reasons incorrectly, but because it outputs reasoning traces (`<think>...</think>`) rather than clean JSON. The environment's `empty_response` guard correctly returns 0.0001 when required fields are absent. **Random Baseline (0.293) beats DeepSeek-R1-32B (0.171)** because a random but structurally valid JSON always provides partial credit vs. malformed output that provides none.
